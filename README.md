# rlab 

Welcome to the central repository for our class R examples. This project is designed to help you bridge the gap between theory and code.

## 📁 What's Inside
- **/data**: Source datasets used in our examples and disk store of objects in computer disk.
- **/**img**: Images used in notebooks.
- **/notebooks**: Completed versions of in-class exercises.
- **/reports**: Store the HTML output files created when knitting.

## 📦 How to Use This
1. **Open the Project**: Double-click the `rlab.Rproj` file to open RStudio with the correct working directory.
2. **Explore**: Open any script in the `/solutions` folder to follow along with the lesson.
3. **Run Code**: Use `Ctrl + Enter` (Windows) or `Cmd + Enter` (Mac) to run lines of code directly from the scripts.

## 🔧 Requirements
To run these examples, make sure you have the following installed:
* [R](https://cloud.r-project.org/)
* [RStudio](https://posit.co/download/rstudio-desktop/)
* Recommended Packages: `tidyverse`
